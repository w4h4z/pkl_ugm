# Manual Installation

1. Download Zip
2. Put in htdocs
3. Rename as pkl_ugm
4. Run XAMPP
5. Open Browser
6. Run localhost/phpmyadmin
7. Import db_sipkldssdi.sql
8. Run localhost/pkl_ugm
9. That's it
